Portfólio 📁

Projeto: Imersão de Dados com Python (Alura)

Este projeto foi desenvolvido durante a Imersão Dados da Alura com foco em análise de dados usando Python e bibliotecas específicas para ciência de dados; o objetivo foi aplicar conceitos práticos de programação e estatística para explorar, tratar e visualizar um conjunto de dados real, transformando informações brutas em insights acionáveis e documentando todo o processo de análise.

Objetivos do projeto:

O projeto teve como objetivos desenvolver habilidades práticas em Python para análise de dados, aprender a manipular e explorar conjuntos de dados com bibliotecas como Pandas e Numpy, criar visualizações informativas com Matplotlib e Seaborn para comunicar insights de forma clara e documentar o processo de análise para compartilhamento em repositório público no GitHub.

Tecnologias utilizadas:

Foram utilizadas como base a linguagem Python e o ambiente Jupyter Notebook para experimentação e documentação; as bibliotecas Pandas e Numpy foram empregadas para limpeza e manipulação de dados, enquanto Matplotlib e Seaborn foram usadas para gerar gráficos e visualizações; arquivos auxiliares do projeto incluem app.py, dados-imersao-final.csv e requirements.txt.

Principais entregas:

As entregas incluem a organização dos dados em tabelas e gráficos que facilitam a interpretação, scripts em Python para automatizar etapas de análise, documentação detalhada em português e inglês no README.md que descreve objetivos e aprendizados, e a publicação do projeto em um repositório público no GitHub para garantir versionamento e transparência.

Aprendizados:

Ao longo do projeto houve fortalecimento da confiança no uso de Python para ciência de dados, prática de boas práticas de versionamento com Git/GitHub, aplicação de conceitos de estatística e visualização para extrair e comunicar insights, e experiência em compartilhar um projeto de forma aberta e colaborativa.
